# ☀️ Solar AI Solutions: Intelligent Proposal Generator

An industry-ready, AI-powered solar proposal generation system and mini-CRM. This project automates the complex calculation of ROI, system sizing, and carbon offsetting, producing a premium-ready PDF proposal for clients in seconds.

## 🚀 Live Demo
[https://solarproposalai.vercel.app]

## ✨ Key Features
- **AI-Powered Proposal Generation:** Uses Google Gemini 1.5 Flash API to generate highly persuasive, structured solar proposals in realtime.
- **Dynamic ROI Visualization:** Interactive 10-year cumulative cost projections using **Chart.js**, visually demonstrating the financial benefits of switching to solar.
- **Premium PDF Layout:** Custom-built "Sun AI Solutions" letterhead system with automatic dating, reference numbers, and professional signature blocks.
- **Built-in Mini CRM:** High-performance dashboard with persistent local storage to track leads, calculate system sizes (kW), and monitor deal status.
- **Advanced Glassmorphism UI:** Modern tech-first design with Plus Jakarta Sans typography and an amber-to-orange gradient aesthetic.

## 🛠️ The Stack
- **Frontend:** Vanilla HTML5, CSS3 (Advanced Flex/Grid & Glassmorphism)
- **Logic:** Vanilla JavaScript (ES6+)
- **Brain:** Google Generative AI (Gemini 1.5 Flash API)
- **DataViz:** Chart.js
- **PDF Engine:** html2pdf.js
- **Persistence:** LocalStorage API

## 📋 How to Use Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/chaitanyashete03/solar-proposal-ai.git
   ```
2. Open `index.html` in any modern web browser.
3. Click the **API Settings** button and enter your [Google AI Studio API Key](https://aistudio.google.com/app/apikey).
4. Add a lead, hit **Generate**, and download your 100% professional solar proposal!

## ⚡ Why this exists?
Manual solar proposal generation typically takes 15–30 minutes per lead. This system reduces that time to **under 15 seconds**, directly improving sales velocity and client conversion for solar engineering teams.
